# 🚚 Transporte "JP Zampa" - Plataforma Web (Frontend)

## 📌 Descripción del Proyecto
Aplicación cliente (Frontend) del **Sistema Inteligente de Gestión Operativa y Financiera**, desarrollado como Proyecto Final de la carrera Ingeniería en Sistemas de Información (UTN FRRo). 

Esta plataforma centraliza la administración de operaciones logísticas, resolviendo la fragmentación documental y proporcionando interfaces de alta usabilidad para la carga de datos de los viajes, liquidaciones y mantenimiento de la flota.

## ⚙️ Arquitectura y Tecnologías
El desarrollo está estructurado bajo una arquitectura basada en componentes, garantizando escalabilidad y un mantenimiento ágil del código:
* **Framework:** Angular
* **Lenguaje:** TypeScript
* **Estilos:** CSS / SCSS (con diseño responsivo)
* **Despliegue:** Vercel (Proyectado)

## 🚀 Módulos Principales de la Interfaz
* **Panel de Control (Dashboard):** Visualización en tiempo real del Índice de Eficiencia del Viaje (IEV) y métricas de flujo de caja.
* **Gestión de Viajes:** Interfaces dinámicas para la asignación de choferes, camiones y destinos.
* **Carga Documental:** Módulo de captura y subida de comprobantes de gastos y peajes, preparado para interactuar con los servicios de visión computacional del backend.
* **Mantenimiento:** Pantallas de control y alertas para el seguimiento del estado mecánico de las unidades.
